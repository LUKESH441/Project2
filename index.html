About the Project

Project Purpose

This is a student data analytics project aimed at understanding modern digital habits. The goal is to analyze patterns in social media consumption, identifying demographic trends, preferred platforms, and potential correlations in regional usage.

Methodology

The dataset used in this analysis is a generated sample mirroring real-world distributions. The data processing and core analysis were mocked conceptualizing typical workflows utilizing modern data structures.
